# Udrink

## Objectif

Il s'agit d'une application en 3 tiers qui permettra à un utilisateur authentifié de récupérer des recettes de coktails à partir de sa liste d'ingrédients.

## Architecture

- ### 1 - Frontend
VueJS.
- ### 2 - Backend
DjangoRest.

- ### 3 - Base de donnée

Base de donnée MySql.
